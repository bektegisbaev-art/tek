# Тез баптау — APK-ны GitHub арқылы құрастыру

Бұл — Android Studio керек етпейтін жоба. GitHub-қа жүктесеңіз, ол бұлтта
APK-ны автоматты құрастырып, жүктеп алуға береді.

## Қадамдар

1. **GitHub аккаунт** ашыңыз (тегін): https://github.com → Sign up.

2. **Жаңа репозиторий** жасаңыз:
   - Жоғарыдан "+" → **New repository**.
   - Атын жазыңыз (мысалы `tez-baptau`).
   - **Public** қалдырыңыз → **Create repository**.

3. **Файлдарды жүктеңіз:**
   - Жаңа репозиторий бетінде **"uploading an existing file"** сілтемесін басыңыз.
   - Осы ZIP ішіндегі **барлық файл мен қалтаны** (`.github`, `app`, `build.gradle`,
     `settings.gradle`, `gradle.properties`, `.gitignore`) сүйреп тастаңыз.
   - Төменде **Commit changes** басыңыз.

4. **Құрастыру автоматты басталады.** Жоғарыдан **Actions** қойындысын ашыңыз —
   "Build APK" деген жұмыс жүріп жатады (1–3 минут).

5. **APK-ны жүктеп алыңыз:**
   - Жұмыс жасыл белгімен біткен соң, оны басыңыз.
   - Төменде **Artifacts** бөлімінен **TezBaptau-apk** файлын жүктеп алыңыз.
   - Ішінен `app-debug.apk` шығады.

6. **Телефонға орнатыңыз:**
   - APK-ны телефонға көшіріп, басыңыз.
   - "Белгісіз көздерден орнатуға" рұқсат сұраса — рұқсат беріңіз.
   - Орнатылды!

## Ескерту

Қосымша root-сыз болғандықтан, авиарежим мен GPS-ті өздігінен өшіре алмайды —
ол тек керекті баптау бетін бір рет түртіп ашуға мүмкіндік береді. Бұл —
Android жүйесінің шектеуі, кодтың кемшілігі емес.
